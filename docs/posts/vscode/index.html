工欲善其事，必先利其器。

快捷键

CK

C-K C-F为格式化当前段落 (Format as F) C-K C-T is to choose theme

CS

C+S-I is to format all the doc C+S-K is to kill the current line C+S+[ is to fold

S

S+A is multi-cursor

C

C-SPACE is intellisense C+[ is to tab the current line.

F

F2 is to rename symbol F8 is to detect errors
