逗号的用法

1if()
2{
3    AAA;
4    BBB;
5    CCC;
6    return DDD;
7}

可以改成

1if()
2{
3    return AAA,BBB,CCC,DDD;
4}

3 Link to heading

赋值语句和普通表达式的值

1A=func(B);
2printf("%d",A);

可以改为

1printf("%d", A=func(B));

4 Link to heading

短路语句

1if(AAA)
2	BBB

可以写成

1(!AAA)||BBB;

当AAA为假，则短路，不计算BBB

当AAA为真，计算BBB
